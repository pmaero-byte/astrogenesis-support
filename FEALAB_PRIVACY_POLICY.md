## Overview

FEA Lab is a finite element analysis application for macOS. This privacy policy explains what data FEA Lab does and does not collect, and how the application accesses your device.

**Summary: FEA Lab collects no personal data, no usage data, and no telemetry of any kind. All computation runs entirely on your device.**

---

## 1. Data Collection

FEA Lab does **not** collect, store, transmit, or share any personal data.

Specifically, FEA Lab does **not**:

- Create or require user accounts
- Collect your name, email address, or any other personal identifiers
- Record or transmit analytics or usage statistics
- Use crash-reporting services that send data off-device
- Implement any form of telemetry or behavioral tracking
- Use advertising SDKs or third-party tracking frameworks
- Store any data on external servers operated by the developer or third parties

---

## 2. On-Device Computation

All finite element analysis computations performed by FEA Lab run entirely on your local machine. No model data, mesh data, simulation inputs, or simulation results are ever transmitted to any server or cloud service. Your engineering data stays on your device at all times.

---

## 3. File System Access

FEA Lab accesses files on your file system only when you explicitly select them. This includes:

- Opening project files or model files you choose via the standard macOS Open dialog
- Saving or exporting results to locations you specify via the macOS Save dialog

FEA Lab operates within the macOS App Sandbox. It cannot access files outside the locations you explicitly grant access to, and it does not scan, index, or read any files without your direct action.

---

## 4. Network Access

FEA Lab has no network functionality except for one optional feature: the **Help > Documentation** menu item, which opens an in-app web view or your default browser to load the FEA Lab documentation page hosted by the developer.

This network request:

- Is initiated only when you explicitly click **Help > Documentation**
- Fetches documentation content from the developer's website
- Does not transmit any personal data, device identifiers, or usage information as part of the request
- Can be avoided entirely by simply not using the Help > Documentation feature

No other network connections are made by FEA Lab under any circumstances.

---

## 5. macOS App Sandbox

FEA Lab is a sandboxed macOS application distributed through the Apple Mac App Store. The macOS App Sandbox restricts FEA Lab to a limited set of system resources and enforces the file system access restrictions described above. FEA Lab does not request entitlements beyond what is required for its core finite element analysis functionality.

---

## 6. Third-Party SDKs and Services

FEA Lab does not integrate any third-party SDKs for analytics, advertising, crash reporting, or data collection. The application contains no code from third-party services such as Firebase, Crashlytics, Amplitude, Mixpanel, or similar platforms.

---

## 7. Children's Privacy

Because FEA Lab does not collect any data from any user, it presents no special privacy risk to children. FEA Lab is a professional engineering tool and is not directed at children under 13.
